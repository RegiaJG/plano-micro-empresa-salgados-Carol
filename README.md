# 🥟 Business Plan — Snacks & Sweets Micro Business

An interactive, single-file business plan for launching a home-based micro business specializing in Brazilian artisan snacks (*salgados*) and sweets (*brigadeiros*). Covers everything from legal registration to paid traffic strategy — no frameworks, no dependencies.

---

## ✨ Features

- **18 navigable pages** with a fixed sidebar index
- Navigate by click, prev/next buttons or **← → arrow keys**
- Progress bar tracking current position
- Responsive layout (mobile + desktop)
- Interactive checklists per step
- Detailed budget with revenue projection

## 🚀 Deploy

Static HTML — no build step, no dependencies.

### Vercel (recommended)

1. Fork or clone this repository
2. Go to [vercel.com](https://vercel.com) and import the repository
3. Keep all default settings — `vercel.json` handles routing
4. Click **Deploy**

### GitHub Pages

1. Go to **Settings → Pages**
2. Source: `Deploy from a branch` → branch `main` → folder `/root`
3. Access via `https://your-username.github.io/repo-name/plano-micro-empresa-salgados.html`

### Local

Open `plano-micro-empresa-salgados.html` directly in any browser — no server required.
